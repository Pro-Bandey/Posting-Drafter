# Next.js + Supabase Draft Manager

A modern draft manager application built with **Next.js App Router**, **Supabase**, and **Tailwind CSS**.

## Features

- Upload image/video drafts
- Store media in Supabase Storage
- Save title, caption, and media URL in Supabase database
- Preview drafts (image/video)
- Delete drafts (database + storage)
- Responsive and modern UI
- Client-side validation and error handling
- Loading states and progress feedback

## Tech Stack

- Next.js 14 (App Router)
- Supabase (Database + Storage)
- Tailwind CSS
- TypeScript
- clsx (conditional classNames)
